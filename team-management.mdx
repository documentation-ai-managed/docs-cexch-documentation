## Team Setup

Effective team management is key to successful project execution. Cexch provides tools to invite members, assign roles, and foster collaboration.

<Steps>
  <Step title="Invite Members" icon="user-plus">
    Send invitations to team members via email.
  </Step>
  <Step title="Assign Roles" icon="shield">
    Set appropriate permissions for each member.
  </Step>
  <Step title="Create Groups" icon="users">
    Organize members into functional groups.
  </Step>
  <Step title="Monitor Activity" icon="activity">
    Track team productivity and engagement.
  </Step>

## Team Communication

<Expandable title="Built-in Chat" default-open="false">
  Use Cexch's integrated chat for quick team communication.
</Expandable>

<CodeGroup tabs="JavaScript,Python">
  ```javascript
const teamMember = {
  email: 'member@example.com',
  role: 'member',
  projects: ['project1', 'project2'],
};

const response = await fetch('/api/team/invite', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer your_token',
  },
  body: JSON.stringify(teamMember),
});
```
  ```python
import requests

team_member = {
    'email': 'member@example.com',
    'role': 'member',
    'projects': ['project1', 'project2'],
}

response = requests.post('/api/team/invite', json=team_member, headers={
    'Authorization': 'Bearer your_token',
})
```
</CodeGroup>
